# Fibonacci Series

## Answer

Print Fibonacci sequence.

---

## Explanation

In Fibonacci series:
```js
Next Number = Previous + Current
```

Sequence starts from:
```js
0 1
```

Every next number is generated by adding:
- previous number
- current number

---

## Brute Force Approach

```js
function fibonacci(n) {

  if (n <= 1) {

    return n;
  }

  return fibonacci(n - 1) +
         fibonacci(n - 2);
}

for (let i = 0; i < 7; i++) {

  console.log(fibonacci(i));
}
```

---

## Output

```js
0
1
1
2
3
5
8
```

---

## Optimized Approach

```js
let prev = 0;

let curr = 1;

const n = 5;

console.log(prev);

console.log(curr);

let i = 0;

while (i < n) {

  let next = prev + curr;

  prev = curr;

  curr = next;

  console.log(next);

  i++;
}
```

---

## Output

```js
0
1
1
2
3
5
8
```

---

## Time Complexity

```js
O(n)
```

# Factorial

## Answer

Find factorial of a number.

---

## Explanation

Factorial means:
```js
Multiplication of all numbers
from 1 to n
```

Example:
```js
5! = 5 * 4 * 3 * 2 * 1
```

---

## Brute Force Approach

```js
function factorial(n) {

  if (n === 0 || n === 1) {

    return 1;
  }

  return n * factorial(n - 1);
}

console.log(factorial(5));
```

---

## Output

```js
120
```

---

## Optimized Approach

```js
function factorial(n) {

  let result = 1;

  for (let i = 1; i <= n; i++) {

    result *= i;
  }

  return result;
}

console.log(factorial(5));
```

---

## Output

```js
120
```

---

## Time Complexity

```js
O(n)
```

# Prime Number

## Answer

Check whether number is prime or not.

---

## Explanation

A prime number is divisible by:
```js
1 and itself only
```

Example:
```js
2, 3, 5, 7
```

---

## Brute Force Approach

```js
const num = 7;

let isPrime = true;

for (let i = 2; i < num; i++) {

  if (num % i === 0) {

    isPrime = false;

    break;
  }
}

console.log(isPrime);
```

---

## Output

```js
true
```

---

## Optimized Approach

```js
const num = 7;

let isPrime = true;

for (let i = 2; i <= Math.sqrt(num); i++) {

  if (num % i === 0) {

    isPrime = false;

    break;
  }
}

console.log(isPrime);
```

---

## Output

```js
true
```

---

## Time Complexity

```js
O(√n)
```

# Armstrong Number

## Answer

Check whether a number is Armstrong number or not.

---

## Explanation

An Armstrong number is a number where:
```js
Sum of digits raised to power
of total digits equals original number
```

Example:
```js
153

1³ + 5³ + 3³ = 153
```

---

## Brute Force Approach

```js
const num = 153;

const str = num.toString();

let sum = 0;

for (let i = 0; i < str.length; i++) {

  sum += Math.pow(
    Number(str[i]),
    str.length
  );
}

console.log(sum === num);
```

---

## Output

```js
true
```

---

## Optimized Approach

```js
let num = 153;

let original = num;

let sum = 0;

while (num > 0) {

  let digit = num % 10;

  sum += digit ** 3;

  num = Math.floor(num / 10);
}

console.log(sum === original);
```

---

## Output

```js
true
```

---

## Time Complexity

```js
O(log n)
```
